**Dev Party Quest** は、プログラミング学習者向けの、RPG風・開発チームロール診断アプリです。

8つの質問に答えることで、自分が開発パーティーの中でどのような役割を担いやすいかを診断できます。

結果は「プロダクト勇者」「UI魔法使い」「ロジック剣士」など、RPGの職業風に表示されます。イベント内で気軽に遊べて、結果をXで共有できるミニアプリを目指します。

## アプリ名

**Dev Party Quest**

## サブタイトル

あなたの開発パーティーロール診断

## 想定ユーザー

- プログラミング学習者
- チーム開発に興味がある人
- ミニアプリイベントで気軽に診断を楽しみたい人

## コンセプト

開発チームをRPGのパーティーに見立て、ユーザーの開発スタイルや得意な立ち回りを8タイプに分類します。

診断結果では、ユーザーの強みやチームでの立ち回りをポジティブに表示し、自己理解とイベント内での交流のきっかけを作ります。

## 使用技術

- React
- TypeScript
- Tailwind CSS

## デプロイ

- Vercel
- 公開URL: https://devpartyquest.vercel.app/

## 主な機能

- トップページ表示
- 診断開始
- 質問表示
- 回答選択
- 進捗表示
- スコア計算
- 結果表示
- もう一度診断する
- Xで共有

## 診断タイプ

| タイプ名 | RPG職業名 | 役割 |
| --- | --- | --- |
| プロダクト勇者 | 勇者 | ユーザー価値を考えて前に進める |
| UI魔法使い | 魔法使い | 見た目・体験づくりが得意 |
| ロジック剣士 | 剣士 | 処理や実装を着実に組む |
| バグハンター | 狩人 | エラーや違和感を見つける |
| 設計参謀 | 軍師 | 構造化・設計・整理が得意 |
| コミュ力僧侶 | 僧侶 | チームを支え、相談しやすい空気を作る |
| 爆速忍者 | 忍者 | とりあえず動かすスピード型 |
| 改善鍛冶師 | 鍛冶師 | リファクタ・磨き込みが得意 |

## バックエンドについて

本アプリでは、バックエンドは使用しません。

認証・保存機能は実装せず、質問データ・結果データはフロントエンド側の定数として管理します。

## 開発メモ

詳細な設計、診断仕様、データ設計、実装計画は `.codex/Plan.md` にまとめます。

`.codex/TODO.md` は、`.codex/Plan.md` を仕様の正としたうえで、実装者が順番に進められるIssue一覧として管理します。実装作業では `TODO.md` のIssue順に進め、仕様の判断に迷った場合は `Plan.md` を優先します。
